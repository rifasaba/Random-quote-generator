Project Name : Random Quote Generator

What it does :
This project fetches a random quote from the DummyJSON API
(https://dummyjson.com/quotes/random)
and displays the quote along with the author on the screen.
Every time the user clicks the button, a new random quote is shown.

How to run it :
Run npm install to install dependencies
Start the project using terminal or command prompt : npm run dev
